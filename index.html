About PlayHub

PlayHub is a personal collection of simple, browser-based learning games designed to support early numeracy, reading, and logical thinking.
The games are intentionally lightweight, easy to understand, and suitable for young children, with parents and educators in mind.

Privacy Note

PlayHub does not collect personal information, require user accounts, or track users.
All games run directly in the web browser.

Educational Use

These games are provided free for personal and educational use.
